Description

This website was made for a final project in my website development class in November 2022. The site is built off the Bootstrap 5 framework and is fully responsive regarding screen width and height. The "Weather" tab makes use of the TomTom API. It allows you to run a "Fuzzy search" which returns to location based on the string you input. That location is then passed to the OpenWeather API and is used to get a five-day forecast of that location. Once this is complete, the result is stored in an SQL database hosted on this server, and you can use the "History" tab to search for past queries. Overall, I am pleased with this project, and it came together very well in the end. Looking back, I would really like to improve the color scheme and add more content on the home page if I could do it again.
